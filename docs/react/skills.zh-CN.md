本指南介绍 Ant Design 的技能包，帮助 AI 工具和智能体工作流更好地理解与使用组件能力。

## 什么是 SKILL.md？ {#what-is-skill-md}

`SKILL.md` 用于定义技能包。本仓库在 `.skills/components/SKILL.md` 中提供 `antd-components` 技能包，每个组件的引用文档位于 `.skills/components/references`。

## 如何使用 {#how-to-use-skill}

1. 将 `.skills/components/SKILL.md` 加入工具的技能注册或上下文。
2. 使用具体组件时，从 `.skills/components/references/Component.md` 加载对应引用文件。

## 组件 Skill 列表 {#component-skill-list}

下表列出当前支持的组件 Skill，并提供可复制的命令。复制后执行该命令会从 GitHub 拉取对应组件的引用文档，并将 Markdown 内容复制到剪贴板。

> 提示：命令默认使用 macOS 的 `pbcopy`。 Windows 请替换为 `clip`， Linux请替换为 `xclip -selection clipboard`。
