## RFC 与 Issue 的区别

| 维度 | RFC (Discussion) | Issue |
|---|---|---|
| 用途 | 征求意见、讨论方案 | Bug 报告、功能请求 |
| 阶段 | 实现前 | 实现中或已确定方案 |
| 格式 | 自由格式，建议结构化 | 使用模板 |
| 关闭 | 标记状态后保持开放 | 解决后关闭 |
