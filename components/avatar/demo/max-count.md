## zh-CN

使用 HOC 封装 `Avatar.Group`，添加 `overflowInFinal` 属性。开启后 `max.count` 表示总共显示的元素数量，会预留 1 个位置给溢出指示器，`max.count` 默认为 3。

## en-US

Wrap `Avatar.Group` with HOC to add `overflowInFinal` prop. When enabled, `max.count` represents the total number of elements to display and reserves 1 slot for the overflow indicator. `max.count` defaults to 3.
