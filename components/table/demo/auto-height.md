## zh-CN

在高度稳定的容器内，可以封装一个小的 helper 组件来计算表格 body 的 `scroll.y`。请确保外层容器有明确高度，或能通过 flex/grid 解析出稳定高度。

## en-US

When Table needs to fill a container with stable height, you can wrap it with a small helper component to calculate `scroll.y` for the table body. Make sure the outer container has an explicit height, or a stable height resolved by flex/grid.
