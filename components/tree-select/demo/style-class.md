## zh-CN

使用 `classNames` 和 `styles` 属性来自定义 TreeSelect 组件的样式。

- `classNames` 和 `styles` 支持对象和函数两种形式
- 对象形式：直接设置各部分的类名和样式
- 函数形式：`(info: { props }) => object`，可以根据组件的 props 动态设置样式

## en-US

Use `classNames` and `styles` properties to customize the style of TreeSelect component.

- `classNames` and `styles` support both object and function forms
- Object form: directly set class names and styles for each part
- Function form: `(info: { props }) => object`, can dynamically set styles based on component props
