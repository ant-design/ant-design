## zh-CN

`size="small"` 表示小号开关。

## en-US

`size="small"` represents a small sized switch.

````__react
import { Switch } from 'antd';

ReactDOM.render(
  <div>
    <Switch />
    <br />
    <Switch size="small" />
  </div>
, mountNode);
````
