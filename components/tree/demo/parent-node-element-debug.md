## zh-CN

父节点标题出现换行时，统一展开箭头、复选框、标题内容垂直对齐方式，且展开箭头 hover 跟随展开图标位置（大小保持源代码固定大小）。

## en-US

When the parent node title wraps, vertically align the expand arrow, checkbox, and title content; the expand arrow hover area follows the expand icon position while keeping the same fixed dimensions as defined in the source styles.
