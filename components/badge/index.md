# Badge

- category: Components
- chinese: 徽标数
- type: 展示

---

图标右上角的圆形徽标数字。

## 何时使用

一般出现在通知图标或头像的右上角，用于显示需要处理的消息条数，通过醒目视觉形式吸引用户处理。

## API

| 参数           | 说明                             | 类型       |  可选值 | 默认值 |
|----------------|----------------------------------|------------|---------|--------|
| count          | 展示的数字，大于 99 时显示为 99+ | Number     |         |        |
| dot            | 不展示数字，只有一个小红点       | boolean    |         | false  |
