## en-US

Use ribbon badge.

```jsx
import { Badge, Card } from '@allenai/varnish';

ReactDOM.render(
  <Badge.Ribbon text="Pushes open the window">
    <Card>And raises the spyglass.</Card>
  </Badge.Ribbon>,
  mountNode,
);
```
