## zh-CN

展示新的组件 Token 功能，支持为不同类型的通知设置不同的背景色。可以通过 `colorSuccessBgElevated`、`colorErrorBgElevated`、`colorInfoBgElevated`、`colorWarningBgElevated` 来自定义各种类型通知的背景色。

## en-US

Demonstrates the new component token functionality that supports different background colors for different notification types. You can customize the background colors using `colorSuccessBgElevated`, `colorErrorBgElevated`, `colorInfoBgElevated`, and `colorWarningBgElevated` tokens.
