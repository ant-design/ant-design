## zh-CN

通过 `variant` 设置标签外观：`filled`（默认）、`solid`、`outlined` 与 `blur`。`blur` 使用半透明背景与 `backdrop-filter`，呈现毛玻璃效果，适合叠在图片或渐变上。渐变区域仅展示 `blur`；请在下方中性背景上对比四种变体——其余变体依赖主题色，不宜直接叠在任意照片或渐变上。

## en-US

Use `variant` to set the tag appearance: `filled` (default), `solid`, `outlined`, and `blur`. The `blur` variant uses a translucent background with `backdrop-filter` for a frosted-glass look on top of imagery or gradients. The gradient strip only demonstrates `blur` — compare all four variants on the neutral background below; other variants rely on theme tokens and are not intended for arbitrary photo or gradient overlays.
