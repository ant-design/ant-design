## zh-CN

可以通过设置 image 为 'simple' 选择另一种风格的图片。

## en-US

You can choose another style of image by setting image to 'simple'.

```jsx
import { Empty } from 'antd';

ReactDOM.render(
  <Empty image="simple" imageSize={40} />,
  mountNode
);
```
