## zh-CN

可以通过设置 `type` 为 `NO_RESULTS` 选择另一种风格的图片。

## en-US

You can choose another style of `type` by setting image to `NO_RESULTS`.

```jsx
import { Empty } from 'infrad';

ReactDOM.render(<Empty type="NO_RESULTS" />, mountNode);
```
